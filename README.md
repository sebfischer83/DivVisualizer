# DivVisualize

## Warum gibt es das Programm?

Ich liebe Statistiken und ich mag Dividenden, ich nutze diverse Tools doch ich hätte gerne über meine
Dividenden noch die ein oder andere Ansicht. Darum ist die Idee für dieses Programm entstanden.

## Was ist es?

Vorhandene Daten über Dividenden sollen auf mehrere Arten dargestellt werden um eine bessere Übersicht
und eine bessere Vorstellung über die Entwicklung zu haben.

## Was ist es NICHT?

Ich werde nur vorhandene Daten nutzen und nicht weiter anreichern aus anderen Quellen.

## Warum keine Website?

Eine Website würde bedeuten das ich das ganze Zeug mit DSGVO und allen anderen Kram an der Backe hätte,
außerdem geht es hier um die privaten Finanzdaten der Nutzer und darum möchte ich das alles komplett offline nutzbar ist.
Die Daten werden niemals euren PC verlassen!

## Warum dann nur für Windows?

Der Einfachheit halber und weil ich schon die ganze Zeit wieder was mit Blazor machen wollte, setze ich
zurzeit auf die WebView2 auf, doch nehme ich an das sicher in Zukunft nochmal eine Platformunabhängigere Lösung kommen könnte.
